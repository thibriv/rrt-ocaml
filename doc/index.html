Presentation

A RRT solving programm for conflicts resolution on aircrafts.

Installation

Compile using command :

> make

This command creates an executable file prog and an HTML documentation that can be found in subdirectory doc/

Usage

Run the program :

> ./prog

Reference manual

List of modules:
